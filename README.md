# MOSLA Language

A modern, AI-first programming language for building intelligent applications, web services, automation, and native software.

Status: 🚀 Production Ready

## Features
- AI-first standard library
- Built‑in AI code checker & builder
- Full HTTP server with static files and API builder
- Tensors and neural network primitives
- Concurrency and robotics mocks
- Python FFI
- Modern syntax
- Cross‑platform

## Quick Start
```bash
npm install -g mosla-lang
mos new MyProject
cd MyProject
mos run

Command Description
mos new <name> Create a new project
mos run [file] Run a MOSLA file
mos serve [file] Start HTTP server
mos ai-check [file] AI code analysis
mos ai-build [file] AI auto‑fix and build
... and many more See mos --help
